Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language. In this tutorial, beginners can learn how to use this technology to create fonts that are visually appealing and engages their audience through different styles of font and font size.

One of the most important choices you'll make for your website is the font for your written content displayed on your website.

If the visitor does not have the font you have chosen, their browser will use the default font instead. Therefore, choosing multiple font types is highly recommended. To do this, use the font-family property to list more than one font. The browser will use the first listed font that is available on the visitor's system or loaded as a web font. You should include these three categories of fonts:

  • The preferred font
  • One or more alternatives
  • A generic standard font incase your other options do not work on their web browser

To specify for these alternative options of fonts, you would type the font-family: name, with name being your first choice of font. Then type name2 and name3 (your second and third choices), only being seperated by a comma and space. Font Size

The font-size property sets the size of your text. This is another very important part of your design, as not also does the size of text determine where a visitor looks, but also helps a visitor determine what is your heading, a paragraph and so on, ultimately helping them determine what is the most important information to read.

There are two primary ways to change the font-size of your text. These are through pixels, or have the size be relative to the element's parent font size by using a percentage, em.

Font Size With Pixels

To set a font size using pixels, you should use the font-size property within your desired selector. Then, you would indicate the number of pixels you would like that text to be.
